# Pembagian Presentasi

## Orang 1: Konsep, Database, dan Relasi
- Menjelaskan latar belakang Sistem Wardrobe & Outfit Planner.
- Menjelaskan tujuan aplikasi: membantu user mengelola pakaian dan membuat kombinasi outfit.
- Menjelaskan tabel: users, categories, clothes, outfits, outfit_details.
- Menjelaskan relasi: user punya banyak pakaian, user punya banyak outfit, outfit punya banyak pakaian melalui pivot.

## Orang 2: Demo Aplikasi
- Register dan login user.
- CRUD kategori.
- CRUD pakaian dan upload foto.
- Membuat outfit dengan memilih beberapa pakaian.
- Melihat riwayat outfit.
- Login admin dan melihat dashboard total user, pakaian, outfit.
